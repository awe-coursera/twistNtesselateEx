This application is a prototype for experimenting with creating and position shapes
on a WebGL canvas ... using raw WebGL as opposed to more complex libraries such as
Three.js
The canvas is display an initial collection of objects which demonstrate the possibilities
You can add objects to this collection.
To add a new object use the list box and the various sliders provided to select the object type and its properties
Then click on the "Create a new object" button to create the object you have specified.
At the moment there are no edit facilities for e.g. deleting and duplicating objects
Neither are there and facilities for selecting objects interactively, and adjusting the camera
and projection settings.

Adjust the shape properties using the sliders provided
Then click the Create Shape button to create a shape with those properties.
Select a Shape :
X Position - Range from -3 to +3 0
Y Position - Range from -3 to +3 0
Z Position - Range from -3 to +3 0
X Scale Range from 0.1 to 0.5 0
Y Scale Range from 0.1 to 0.5 0
Z Scale Range from 0.1 to 0.5 0
Rotate About X axis - Range from -30 to +30 degrees 0
Rotate About Y axis - Range from -30 to +30 degrees 0
Rotate About Z axis - Range from -30 to +30 degrees 0
